# Sistema Proposto #
# Nome do Sistema: ETE (Educa Tech Enterprise) #

Este projeto propõe o design de uma Plataforma Acadêmica com o objetivo de centralizar e organizar a vida acadêmica de estudantes e professores. O sistema busca resolver o problema da fragmentação das ferramentas acadêmicas, oferecendo um ambiente único para disciplinas, atividades, comunicação e acompanhamento do desempenho.

## Objetivo:
Criar uma solução digital simples e escalável que facilite a organização acadêmica, melhore a comunicação entre alunos e professores e permita o acompanhamento básico do desempenho acadêmico.

## Problema Central:
Atualmente, o uso de múltiplas plataformas desconectadas dificulta a gestão de disciplinas, tarefas e comunicação, causando desorganização e perda de informações.

## Metodologia:

O desenvolvimento do projeto foi planejado utilizando a metodologia Scrum, por se tratar de uma abordagem ágil que facilita a organização das atividades e o desenvolvimento incremental do sistema. Essa metodologia é adequada ao contexto acadêmico, pois permite melhor controle das tarefas e adaptação ao longo do projeto.

**Metodologia:** Scrum
